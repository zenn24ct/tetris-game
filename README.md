## Tetris風 Webゲーム
ブラウザで遊べるシンプルなテトリス風ゲーム。
JavaScript（HTML / CSS / Canvas）を使用して制作した。

### 遊び方

← →：ブロックの移動

↑：回転

↓：1マス落下（ソフトドロップ）

Space：一気に落下（ハードドロップ）

スタートボタン：ゲーム開始

### 機能

ブロックの回転・落下・移動

行が揃うと自動で消去＆スコア加算

NEXT ブロック表示

ゲームオーバー判定
