## Tetris風 Webゲーム
ブラウザで遊べるシンプルなテトリス風ゲーム。
JavaScript（HTML / CSS / Canvas）を使用して制作した。

### 遊び方

← →：ブロックの移動

↑：回転

↓：1マス落下（ソフトドロップ）

Space：一気に落下（ハードドロップ）

スタートボタン：ゲーム開始

### 機能

ブロックの回転・落下・移動

行が揃うと自動で消去＆スコア加算

NEXT ブロック表示

ゲームオーバー判定

# ブロック崩しゲーム

## 概要
シンプルで難易度の高いブロック崩しゲーム

## 特徴
- ライフは1回のみ（ボールを落としたら即ゲームオーバー）
- 10行×12列の大量ブロック（計120個）
- ブロックに耐久値あり（最大5）
- レベルクリアごとに難易度上昇
- パーティクルエフェクト搭載

## 操作方法
- マウス移動でパドル操作
- 矢印キー（← →）でもパドル操作可能

## ゲームルール
- 全ブロックを破壊するとレベルクリア
- レベルが上がるとボール速度とブロック耐久値が増加
- パドル中央で受けるとボーナス点（+5点）
- ブロック破壊で得点（10点×レベル×ブロック耐久値）

## 技術仕様
- HTML5 Canvas
- Vanilla JavaScript
- レスポンシブ対応

## ファイル構成
- 単一HTMLファイルで完結
- 外部ライブラリ不要
